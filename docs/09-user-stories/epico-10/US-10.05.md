## Escopo

* Executar `docker-compose up` e confirmar todos os serviços sobem
* Chamar os 19 endpoints REST e verificar que `api_requests_total` aparece no Prometheus
* Executar `python3 scripts/validate_startup.py` e confirmar sucesso
* Parar a aplicação e verificar log de shutdown no Loki
* Dashboard Grafana exibindo dados reais
* Confirmar que traces estão habilitados: verificar `management.tracing.enabled=true` em `application.properties` ou log de startup
* Documentar evidência em `docs/process/tests-epico-10/observability-evidence.md`
* Marcar G6, G7, G10 como ✅ em `05-backlog.md`

## Critérios de aceite

* [ ] `docker-compose up` → todos os serviços saudáveis
* [ ] `GET /actuator/prometheus` exibe `api_requests_total` para todos os endpoints chamados
* [ ] `python3 scripts/validate_startup.py` → sucesso
* [ ] Log de shutdown no Loki após `docker-compose down`
* [ ] Dashboard Grafana com dados reais visíveis
* [ ] `GET /actuator/health` e traces habilitados: log de startup confirma `otel.sdk.enabled=true` ou Tempo recebendo spans
