## Contexto

Após US-13.01 e US-13.02:
- Artigos novos: `readingTimeMinutes` calculado no momento da criação
- Artigos existentes (pré-migração): `readingTimeMinutes = null` até a primeira leitura via `GET /articles/:slug`

**Fórmula:** `ceil(wordCount / 200)` — mínimo de 1 minuto.

**Cenários de validação:**

| Cenário | Entrada | Valor esperado |
|---|---|---|
| Artigo com 200 palavras | `body` com 200 palavras | `readingTimeMinutes = 1` |
| Artigo com 400 palavras | `body` com 400 palavras | `readingTimeMinutes = 2` |
| Artigo com 0 palavras (body vazio) | `body = ""` | `readingTimeMinutes = 1` (mínimo) |
| Artigo existente com `null` | Primeira leitura via `GET /articles/:slug` | Campo preenchido; segunda leitura retorna mesmo valor |
