## Contexto

Os 4 fluxos adicionais (M6 = MT + 4) dependem de EPIC-12 e EPIC-13 estarem concluídos.

**Fluxos adicionais (4):**

| Fluxo | Sequência | Validação |
|---|---|---|
| **F4 — Soft delete de artigo** | Login → `POST /articles` → `DELETE /articles/:slug` → `GET /articles` | Artigo não aparece na listagem |
| **F5 — Soft delete de comentário** | Login → criar artigo → `POST /comments` → `DELETE /comments/:id` → `GET /comments` | Comentário não aparece |
| **F6 — Reading time** | Login → `POST /articles` (body longo) → `GET /articles/:slug` | Campo `readingTimeMinutes` presente e > 0 |
| **F7 — Comentar e deletar** | Login → criar artigo → `POST /comments` → `DELETE /comments/:id` → `GET /comments` → validar ausência | Ciclo completo de comentários |
