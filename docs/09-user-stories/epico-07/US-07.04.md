## História de usuário

**Como** desenvolvedor responsável pela qualidade,  
**quero** escrever testes de integração para os repositórios JPA,  
**para que** mutações em queries e lógica de persistência sejam detectadas pelo Pitest.

## Contexto

Os repositórios MyBatis existentes (`JpaArticleRepositoryTest`, `JpaCommentRepositoryTest`, `JpaArticleFavoriteRepositoryTest`, `JpaUserRepositoryTest`) serão substituídos por testes JPA após EPIC-05. Esta história cria ou adapta esses testes para os repositórios JPA migrados.

**Classes alvo (`io.spring.infrastructure.*` após EPIC-05):**
- `JpaArticleRepository` (substituirá `MyBatisArticleRepository`)
- `JpaUserRepository` (substituirá `MyBatisUserRepository`)
- `JpaCommentRepository` (substituirá `MyBatisCommentRepository`)
- `JpaArticleFavoriteRepository` (substituirá `MyBatisArticleFavoriteRepository`)

---

## Escopo

* Criar testes `@DataJpaTest` + Testcontainers PostgreSQL para cada repositório JPA
* Reutilizar `DbTestBase` e `TestHelper` existentes em `src/test/java/io/spring/`
* Cobrir: salvar, buscar, deletar, queries com filtros
* Excluir do escopo Pitest: `io.spring.graphql.*` (código gerado DGS)
* Verificar que mutation score de `io.spring.infrastructure.*` ≥ 90% após os testes

## Fora de escopo

* Testes de read services MyBatis — removidos em US-05.05
* Testes de controllers REST — cobertos em EPIC-08

## Critérios de aceite

* [ ] Cada repositório JPA tem testes `@DataJpaTest` + Testcontainers PostgreSQL
* [ ] Operações CRUD básicas testadas para cada repositório
* [ ] `./gradlew test` 100% verde
* [ ] `./gradlew pitest` — score de `io.spring.infrastructure.*` ≥ 90%

## Regras técnicas

* `@DataJpaTest` com `replace = NONE` para usar Testcontainers PostgreSQL real
* Reutilizar `DbTestBase` existente — não criar nova infraestrutura de teste do zero
* Proibido usar H2 — banco de desenvolvimento é PostgreSQL

## Dependências

* US-07.03 concluído
* EPIC-05 concluído (repositórios JPA existentes)

## Resultado esperado

Infrastructure layer com cobertura de mutação adequada. Total score EPIC-07 ≥ 95%.
